# Client_application_database
**База данных MySQL с клиентским приложением на С#**
## Стек технологий:
- MySQL
- С#
## Как запустить проект:
1. Склонируйте репозиторий
2. Установите MySQL Workbench, Microsoft SQL Server, MySql Connector/NET, Microsoft Visual Studio
3. Откройте файл "База данных тренажерного зала.mwb", Database-Connect to Database, пользователь root, пароль 0000
4. Откройте папку "WindowsFormsApp1", файл "WindowsFormsApp1.sln"
## Возможности программы:
**Добавление, редактирование и удаление записей о клиентах, тренерах, занятиях, абонементах тренажерного зала**
**Поиск данных клиентов по ФИО, номеру телефона, e-mail**
**Просмотр активных абонементов**
**Клиенты и тренеры могут просматривать и редактировать личные данные, просматривать список занятий, клиенты могут просматривать историю занятий, свои абонементы**
## Особенности программы:
1. Встроены составные ключи, не позволяющие записать клиента и тренера на несколько занятий в одно и то же время
2. Разделение ролей (администратор, тренер, пользователь)
3. Добавлен триггер, автоматически меняющий статус активности абонемента по окончанию его срока действия
